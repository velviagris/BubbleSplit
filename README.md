# Bubble Split

## About

This is a lightweight application, inspired by Multitasking 'Bubbles' in Android 17 Beta 3 update, and no "Root/Shizuku" needed.

## Features

- Add your favorite apps
- Show bubbles for incoming notifications
- Tap bubble to split screen
- Dismiss bubbles to avoid distractions

## How to use

1. Allow reading "Notification Access"
2. (Optional) Allow reading "Package Usage Stats"
3. Manage Favorite Apps
4. Enable and display bubble

> After setting up BubbleSplit, when new notifications come, tap bubble to split your screen, feel free to reply message or do anything you want!
